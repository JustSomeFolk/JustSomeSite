november stellaris game

our second stellaris game will take place in the first weekend of nov. both the 7th and 8th if possible
it will be a vanilla game, mainly to avoid to many desync errors like last time, but also to try out the new dlc.
the lobby will be up between 11 and 11:30 and the game will start at 12 and go on for 4-5 hours, maybe more depending on the consensus of the players.

the following origins are banned

  1. shattered ring
  2. common ground
  3. hegemon
  4. lost colony
  5. scion*

the following civics are per default banned

  1. fanatic purifiers
  2. determined exterminator
  3. devouring swarm
  4. terravore

guidelines for pvp

if an players overall relative power (RP for short) is below yours (inferior or pathetic), your wargoal against that player should not be conquest (taking their planets and systems), it should instead be something like Subjugation/Hegemon

choparty's stellaris game event (done)

it is finally time for some stellaris

the dates for the game are the 24 and 25 okt.
the idea is that we play for 4 to 5 hours both days of the weekend to ensure that we might get through a whole game and so that we dont burn out by going 8 to 10 hours one day and be completely exhausted the nex.
the games will run from 12 to 16/17, will start hosting 30 minutes before, so that people have time to join up and create their empires, the game will be password protected, the password will be postet on the discord an hour before the game
the main theme of the game is more focused on diplomacy and roleplay rather then the minmaxed optimal strat, but there are still some origins and civics that are banned.
i hope to get enough people together such that we can play a game without any ai empires.

the following origins are banned

  1. shattered ring
  2. common ground
  3. hegemon
  4. lost colony
  5. birch world*
  6. alderson disk
  7. the great ring
  8. interstellar dwellers*
  9. penrose ringworld
  10. scion*
  11. megaflora tree of life*

the ones marked with * can be allowed, but talk with me about it first

the following civics are per default banned

  1. fanatic purifiers
  2. determined exterminator
  3. devouring swarm
  4. terravore

if you really want to play with one of these civics then talk with me first

guidelines for pvp - if an players overall relative power (RP for short) is below yours (inferior or pathetic), your wargoal against that player should not be conquest (taking their planets and systems), it should instead be something like Subjugation/Hegemon

as some might have noticed by now, some of these origins are not vanilla and that is right, we will be using small modpackage that can be found by following this link: https://steamcommunity.com/sharedfiles/filedetails/?id=2234085175

lastly we will be using the justsomefolk discord server during the game, it should be possible to find a link to that on the website
